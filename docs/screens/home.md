# 首页 (Home Screen)

> 设计来源: 墨刀原型 `C:\H3\APP\PNG\index.png`
> 最后更新: 2026-06-03

## 布局

```
┌──────────────────────────────┐
│ [WiFi信号]          [📁]    │ 顶栏 44px
│ 绿=连接 红=断开      文件夹   │ 背景 #0A0A0A
├──────────────────────────────┤
│                              │
│     Camera Live Preview      │ 取景器 (Expanded)
│   连接时: 实时画面(暖金)       │
│   断开时: 黑色 + 相机图标     │
│                              │
├──────────────────────────────┤
│ [📷缩略]   [○快门]   [●录像]│ 拍摄控制 80px
│  44x44      64x64     48x48  │ 背景 #0A0A0A
├──────────────────────────────┤
│   CAMERA        EFFECTS      │ 底部标签 70px
│   (激活)        (滤镜页)      │ 金色高亮 #D89A0F
└──────────────────────────────┘
```

## 控件规格

### 1. WiFi 信号指示 (左上)
| 属性 | 值 |
|------|-----|
| 类型 | 自定义绘制 WiFi 图标 |
| 连接状态 | 3格弧线 + 底点，绿色 #4CAF50 |
| 断开状态 | 1格弧线 + 底点，红色 #F44336 |
| 旁边文字 | "已连接" / "未连接" |
| 点击行为 | 导航到相机连接页 |

### 2. 文件夹按钮 (右上)
| 属性 | 值 |
|------|-----|
| 图标 | `Icons.folder_outlined` |
| 颜色 | 白色 70% 透明度 |
| 尺寸 | 22px |
| 点击行为 | 打开相机照片预览列表 |

### 3. 取景器 (中部)
| 属性 | 值 |
|------|-----|
| 连接时 | 暖金色背景 #D89A0F，显示实时画面 |
| 断开时 | 黑色背景 #0A0A0A，居中显示白色相机轮廓图标 |
| 断开文字 | "相机未连接" + "请连接相机WiFi开始拍摄" |
| 布局 | Expanded，填充剩余空间 |

### 4. 拍摄控制
| 控件 | 尺寸 | 颜色 | 行为 |
|------|------|------|------|
| 缩略图 (左) | 44x44, 圆角6px | 深灰边框 | 显示最后拍摄照片缩略图 |
| 快门 (中) | 64x64, 圆形 | 白色 + 外发光 | 点击触发拍照，按下缩放到90% |
| 录像 (右) | 48x48, 圆形 | 红色 #C62828 | 点击切换录制状态，录制时显示白色方块 |

### 5. 底部标签
| 标签 | 图标 | 状态 |
|------|------|------|
| CAMERA | camera_alt_outlined | 默认激活，金色 #D89A0F |
| EFFECTS | auto_fix_high_outlined | 未激活，白色35% |

## 状态管理

| 状态 | 变量 | 影响 |
|------|------|------|
| WiFi连接 | `_wifiConnected` | WiFi图标颜色、文字、取景器显示 |
| 录制中 | `_isRecording` | 录像按钮样式切换 |
| 最后拍照 | `_lastPhoto` | 缩略图显示内容 |
| 激活标签 | `_activeTab` | 底部标签高亮 |
